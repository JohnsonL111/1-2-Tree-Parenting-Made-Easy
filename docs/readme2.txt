For task manager:
To add a task, click the floating action  button oon the right hand corner.
To edit task title, click on the task name in the list,
To view detail, either click on the child's photo or the child name in the list.
To delete a task, drag the task in the list and swipe to left.

For coin flip:
To manually set the current picker, click the child photo, inside the list, if chose NOBODY, then it will just toss the coin.

for timeout
To change the speed of the timer, click the button at the right top corner.
To see the current speed, it is located below the animation
